## cat
| Command | Description                                                    |
|---------|----------------------------------------------------------------|
| `-n`    | Show line numbers with new lines.                              |
| `nl`    | Show line numbers and ignore new lines without contents.       |
| `tac`   | Display contents of a file in reverse (from end to first).     |
  
```bash
cat -n file1
nl file
tac file1
```
