


## uname
uname > show what is your OS 
uname -r > show kernel release
uname -i > show hardware platform 
uname -a > show all
```bash
uname
uname -r
uname -i
uname -a
```
## man
man is description of command guid for use.
-w > show path of that man file.
-f > short description
```bash
man ls
man -w ls
man -f ls
```
